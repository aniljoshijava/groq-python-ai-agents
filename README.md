?? AnilTechAI: Agentic AI Framework (Groq + Python)
Welcome to the AnilTechAI Agentic Framework! This project demonstrates how to build high-speed, autonomous AI agents using Python and Groq's LPU Inference Engine.

Key Features:
Ultra-Fast Inference: Powered by Groq for real-time AI responses.

Autonomous Agents: Agents that can perform web searches using Google/DuckDuckGo.

Tool Integration: Easily plug in custom Python tools.

Secure Setup: Uses .env for API key management (Security First! ???).

??? Tech Stack
Language: Python 3.10+

AI Engine: Groq (Llama 3 / Mixtral models)

Framework: Agno / Phidata (Agentic Framework)

Cloud: Google Cloud Platform (GCP)

?? Installation & Setup

1.Clone the Repository:
git clone https://github.com/aniljoshijava/groq-python-ai-agents.git
cd groq-python-ai-agents

2.Create a Virtual Environment:
python -m venv venv
source venv/bin/activate  # For Linux/Mac
venv\Scripts\activate     # For Windows

3.Install Dependencies:
pip install -r requirements.txt

4.Environment Variables:
Create a .env file in the root directory and add your Groq API Key:
GROQ_API_KEY=your_actual_api_key_here
??? Usage
python web_search_agent.py
